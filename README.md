# Portfolio Website

A modern, responsive portfolio website for Noor Bab, a Front-End Developer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode**: Toggle between light and dark themes
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Scroll Animations**: Sections fade in as you scroll
- **Modern UI**: Clean, professional design with gradient backgrounds

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)

## Sections

- **Hero**: Introduction and call-to-action
- **About**: Personal background and motivation
- **Skills**: Technical and soft skills
- **Projects**: Showcase of key projects with descriptions
- **Contact**: Contact information and social links

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Or use a local server for better experience
